# Contributing

Contributions are welcome when they improve skill clarity, installation, QA checks, or prompt robustness.

Please keep changes aligned with the existing LS / LS-Q visual identity:

- Keep LS-Q as the default v2 style.
- Keep legacy flat-style behavior inside `ls-corgi-illustrations`.
- Do not add private article content, client examples, credentials, or unpublished brand materials.
- Do not replace the reference IP images with unrelated generated characters.
- Keep prompts concise enough for practical use.

Before submitting changes, check:

- `SKILL.md` frontmatter is valid.
- All referenced files exist.
- No local absolute paths are present.
- No tokens, API keys, private links, or client materials are present.
