# 柯基 IP

## 角色定义

柯基是个人中文正文配图的固定视觉 IP。

默认每张图都要出现柯基。它是这套正文配图的个人 IP 主角：萌、亲和、回应感强，适合陪读者理解内容。它可以可爱，可以有表情和动作，但画面重点必须是“它正在帮内容运转”，不是单纯卖萌。它没有尾巴。

## 参考图

参考图在 `assets/ip-reference/`：

- `corgi-front.png`：正面坐姿，完整外形和配色参考。
- `corgi-elegant.png`：端杯微笑，亲和与稳定气质参考。
- `corgi-online.png`：上线回应感，活泼和行动感参考。

提取外形特征和角色气质，可以保留参考图的萌感、圆润和热情回应感；不要复刻原图黑色背景、大字标题或 3D 渲染质感。

## 外形

- 橙白柯基，小短腿，身体圆润，整体偏萌系亲和。
- 两只很大的直立三角耳，耳内浅粉色。
- 脸部有从额头到鼻梁的白色中线。
- 黑色圆鼻子，棕黑圆眼，眉毛短而明显。
- 白色胸口和浅奶油色嘴部、爪子。
- 没有尾巴。侧面、背面、跑动、转身时也不要出现尾巴、短尾、卷尾或尾巴轮廓。
- 可以保留小舌头、眨眼、挥爪、歪头等动作，但不要每张都用同一个表情。
- 画成清爽手绘正文插画里的角色：线稿为主，橙白色用于角色识别，整体柔和、圆润、有亲和力。

## 性格

- 热情、开心、接得住，有“收到、我来帮你”的回应感。
- 有点小笨拙，但不蠢，像在努力帮忙。
- 亲和、可爱、陪伴感强，可靠但不严肃。
- 可以眨眼、吐舌、挥爪、歪头，但动作要服务内容。
- 像在正文插画里负责搬运、分拣、提醒、接力、搭桥和把事情变简单。

## 常见职责

让柯基承担核心动作：

- 叼着线头把信息接上。
- 用短腿推一个小机器运转。
- 把散落素材扒拉进纸箱。
- 趴在流程断点上当临时桥。
- 叼着证据卡片铺成信任路径。
- 用爪子按下“判断”按钮。
- 从旧抽屉里刨出可用信息。
- 把混乱便签分到不同篮子。
- 举小牌提醒风险或下一步。
- 守在入口，把无关信息拦下来。
- 拖着一条橙色路径把读者带到下一段。

## 禁止

- 不要生成 3D 渲染、毛绒玩具、纯贴纸包或头像。
- 不要画成低幼儿童卡通海报。
- 不要让柯基只是站在角落里卖萌。
- 不要每张都吐舌、眨眼、挥手，要根据内容换动作。
- 不要加复杂服装、品牌 Logo、项圈铭牌或拟人套装。
- 不要画尾巴。
- 不要让角色抢走结构表达。
- 不要把背景画成宠物场景、狗窝、公园或宠物店，除非正文主题需要。

## 判断标准

如果去掉柯基，图的核心隐喻还能完全成立，说明柯基太装饰了；要重写提示词，让柯基成为动作主体。
