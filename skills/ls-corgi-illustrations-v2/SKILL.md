# LS-Q 柯基正文配图（v2 / Q 版 3D 卖萌风）

## 核心定位

为中文文章设计和生成 16:9 横版正文配图。把文章里的关键判断、流程、结构、状态或隐喻，变成一张可爱、活泼、有立体感的 Q 版 3D 卡通插画。

默认视觉 IP 是 **LS-Q**：一只 Q 版柯基犬，暖橙色主体、米白面部、三角立耳、大圆眼、黑小圆鼻、微笑露小白牙。画风是扁平化 3D 卡通——色块干净但有立体光影、明暗渐变、柔和体积感。

LS-Q 是"可爱又靠谱的小助手"。会抱东西、举牌子、坐姿持咖啡杯、穿魔法袍挥魔法棒——卖萌但不喧宾夺主，始终在做解释画面的核心动作。

## 与 v1 的区别

- **v1（ls-corgi-illustrations）**：扁平色块、温暖不卖萌、不穿服装、不做 3D。稳重的成人草图感。当你明确说"扁平风/用老版"时触发。
- **v2（本 skill，默认）**：Q 版大头比例、扁平化 3D 卡通、立体光影、可爱卖萌、可服装可持物。

## 先读这些参考

按任务需要读取，不要一次塞满上下文：

- `references/style-dna.md`：风格 DNA、颜色语义、文字、禁忌。
- `references/ls-ip.md`：LS-Q 柯基的形象、性格、动作库和禁忌。
- `references/composition-patterns.md`：结构类型、原创隐喻方法和反复刻规则（与 v1 共用，不变）。
- `references/prompt-template.md`：单张生图提示词模板。
- `references/qa-checklist.md`：生成后检查和迭代规则。
- `assets/reference-ip/`：LS-Q 柯基的原始 IP 校准图（3 张样例），用于锁定外形与画风，**不要直接照抄这些图的场景或构图**。
- `assets/examples/`：低频视觉校准。不要照抄构图、物件或标注。

## 工作流

### 1. 消化正文

先读用户给的正文、链接、Notion 页面、Markdown 文件或截图内容。提炼：

- 核心观点是什么
- 哪些段落承担认知转折
- 哪些内容适合用图解释
- 哪些地方只适合文字,不需要图

不要平均配图。优先选择"认知锚点"，例如：核心判断、两个断点、输入输出闭环、分流、前后对比、一鱼多吃、承接路径、常见坑、角色状态变化。

### 2. 先出配图策略

如果用户只是说"分析怎么配图 / 思考哪些地方需要配图"，先给 shot list。每张图写清楚：

- 放在哪个段落后
- 图的主题
- 核心意思
- 结构类型
- LS-Q 在图里做什么
- 建议元素
- 建议中文标注词

默认 4-8 张。文章很短时 1-3 张;长文也不要轻易超过 9 张。够用就好,避免把正文做成画册。

### 3. 单张生成

如果用户明确要求"生成 / 输出 / 做图 / 帮我生成",不要停下来等确认;用内置 `image_gen` 每张单独生成。不要把多张图拼在一张里。

每张图只讲一个核心结构。提示词必须包含：

- 16:9 横版中文正文配图
- 纯白或极浅色背景
- LS-Q 柯基 Q 版 3D 卡通风格（大头比例、暖橙+米白、V 形白纹、三角立耳、大圆眼、黑小圆鼻、微笑）
- 立体光影、明暗渐变、柔和体积感
- 少量红色/橙色/蓝色中文手写批注
- 大量留白
- LS-Q 作为核心动作主体
- 可选服装（魔法袍、围裙、工作马甲等）和道具（咖啡杯、魔法棒、放大镜、键盘等）
- 禁止 PPT、商业精致插画、复杂架构、左上角类型标题

不要复刻过往案例。每次都要从当前文章重新发明一个可爱但成立的隐喻。

### 4. 检查与迭代

生成后检查 `references/qa-checklist.md`。如果出现以下问题,优先重生成或局部编辑：

- LS-Q 只是装饰
- 画面太满
- 太像流程图/PPT
- 中文太多或错字严重
- 左上角出现"常见坑/流程图/系统架构图"等标题
- 画风太冷淡、不够可爱
- 背景不是干净浅色底
- LS-Q 外形漂移（颜色、耳朵、白纹、比例不对）
- 服装道具堆叠过多

### 5. 保存交付

如果用户在 workspace 内工作，把最终图复制到：

```text
assets/<article-slug>-illustrations/
```

按顺序命名：

```text
01-topic-name.png
02-topic-name.png
```

保留原始生成文件，不要覆盖已有资产，除非用户明确要求替换。

## 输出口径

生成前的策略输出要短而准。生成后的交付要包含：

- 生成了几张
- 每张图的用途
- 保存路径
- 哪些图最稳，哪些图是可选

不要长篇解释风格理论；让图自己说话。
