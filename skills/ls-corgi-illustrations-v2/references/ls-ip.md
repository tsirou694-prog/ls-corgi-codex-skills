# LS-Q 柯基 IP

## 角色定义

LS-Q 是中文正文配图的固定视觉 IP。一只 Q 版柯基犬,扁平化 3D 卡通风格。

默认每张图都要出现 LS-Q。LS-Q 既是可爱的视觉锚点,也是解释概念的动作主体——抱、举、卡、按、扛、记录、修补,每张图它都在做"解释这个概念"的那件事。

## 外形（必须严格一致，参考 `assets/reference-ip/`）

### 身体比例

- Q 版大头:头身比约 1:1.2 ~ 1:1.5（头部明显偏大）
- 短腿、矮胖、敦实
- 整体圆润、无棱角

### 毛色分布

- 主体:暖橙色（焦糖橘黄、姜黄色）
- 面部:额头白色 V 形或闪电形竖纹（从鼻梁延伸到头顶）
- 脸颊:两侧白色斑块（从眼下方延伸到嘴角）
- 嘴部周围、下巴、腹部、爪尖:米白色
- 内耳:浅粉色（与外耳橙色形成对比）

### 耳朵

- 三角形立耳,耳尖略圆
- 自然竖立,耳尖微微向前倾斜
- 耳朵比例比头略大

### 眼睛

- 圆形大眼,眼眶偏大
- 深棕色瞳孔接近黑色
- 眼周浅棕色眼影晕染（增加立体感）
- 眼神明亮、带笑意

### 鼻子

- 黑色圆形小纽扣
- 略微圆润饱满

### 嘴部

- 微笑状,嘴角上扬
- 可露少量白色小门牙（2 颗）
- 开心时可露小粉色舌头

### 画风

- 扁平化 3D 卡通:色块干净 + 明暗渐变 + 柔和立体光影
- 线条圆润流畅,无尖锐棱角
- 不要做超写实 3D 渲染（不要毛发颗粒、不要照片级材质、不要复杂光影）
- 也不要纯 2D 平面（要有体积感、光影、立体感）

## 服装与道具（可选，按场景）

LS-Q 默认裸身（仅毛色）。需要表达特定场景时可穿服装或持物：

- **服装**:黑色魔法袍+橙色领巾、白色围裙、工作马甲、咖啡师小围裙等。服装要简单不堆叠。
- **道具**:咖啡杯、魔法棒（棒尖星星特效）、放大镜、望远镜、提示牌、键盘、笔、笔记本。

服装和道具要服务画面意思,不要为装饰而装饰。

## 性格

- 可爱、活泼、温暖
- 卖萌但有目的——萌感服务画面,不喧宾夺主
- 像一个努力又讨喜的小助手
- 会自己上手干活,不是只在那看着

## 常见职责

让 LS-Q 承担核心动作。它的卖萌应该服务核心意思。

- 抱着、扛着、推着、拉着素材（可能抱不动、推不动）。
- 举着牌子、提示牌、警告牌（可能举得歪歪的）。
- 卡在断点、瓶颈、缝隙里（短腿够不到、身体卡住）。
- 在机器旁边按按钮、拉杆杆、拧螺丝（可能姿势别扭）。
- 把信息源拉到一起、把分流通道对齐。
- 站在筛选漏斗旁分拣、挑选。
- 牵着路径线、跟着箭头走（可能走错路或走慢了）。
- 在井里、洞里、纸箱里探出头（够不到边缘）。
- 举着放大镜、望远镜、手电筒看远处。
- 坐姿喝咖啡、敲键盘、记笔记。
- 穿魔法袍挥魔法棒（用于"上线啦/魔法生效/激活"主题）。

## 禁止

- 不要漂移外形:颜色不能变棕或灰、耳朵不能耷拉、面部白纹不能消失、比例不能从 Q 版大头变正常柯基比例。
- 不要做成超写实 3D 渲染（毛发颗粒、照片材质、复杂光影）。
- 不要做成纯 2D 平面（没有立体感、没有光影）。
- 不要过度堆砌服装配饰（最多 1-2 件服装 + 1 个道具）。
- 不要让 LS-Q 只是站在角落里看。
- 不要让 LS-Q 抢走结构表达。
- 不要把 LS-Q 画成冷酷、阴郁、严肃表情。

## 判断标准

如果去掉 LS-Q,图的核心隐喻还能完全成立,说明 LS-Q 太装饰了;要重写提示词,让 LS-Q 成为动作主体。

LS-Q 必须正在做"解释这个概念"的那个动作:搬、举、卡、拉、推、扛、守、按、分拣、记录、修补、挥魔法棒、敲键盘、喝咖啡。如果只是微笑站着,就不合格。
