# LS 柯基 IP

## 角色定义

LS 是中文正文配图的固定视觉 IP。一只橙黄色柯基犬,扁平色块卡通风格。

默认每张图都要出现 LS。LS 不是吉祥物,不是贴纸,不是可爱装饰,而是正在笨拙但认真地参与系统运转的小助手。

## 外形

- 身体主色:橙黄色(温暖、明亮的浅橙黄,参考 `assets/reference-ip/`)。
- 面部有白色条纹,从额头中央延伸到鼻梁。
- 腹部、爪子尖、嘴部周围是白色。
- 黑色小鼻子,鼻头略圆。
- 立耳,耳尖略圆,耳朵比头略大。
- 短腿,身体比例偏长偏矮(典型柯基体型)。
- 圆眼睛,黑色瞳孔,眼神温和专注。
- 嘴角微微上扬,默认带一点微笑。
- 整体扁平色块卡通画风,色块边缘干净,线条简洁。
- 轮廓可以略微手绘抖动,但不要做成 3D 渲染或精致矢量。

## 性格

- 笨拙但靠谱:做的事情偶尔会出错或不够漂亮,但目标是清楚的,最后能把事办成。
- 温暖、认真、不卖萌。
- 像一个有点紧张但很努力的小助手。
- 不冷幽默,不冷漠,不阴郁。
- 会自己上手干活,不是只在那看着。
- 短腿、小体型会带来天然的喜剧感,但不要做成儿童卡通。

## 常见职责

让 LS 承担核心动作。它的笨拙应该服务核心意思,不要为笨拙而笨拙。

- 抱着、扛着、推着、拉着素材(可能抱不动、推不动)。
- 举着牌子、提示牌、警告牌(可能举得歪歪的)。
- 卡在断点、瓶颈、缝隙里(短腿够不到、身体卡住)。
- 在机器旁边按按钮、拉杆杆、拧螺丝(可能姿势别扭)。
- 把信息源拉到一起、把分流通道对齐。
- 站在筛选漏斗旁分拣、挑选。
- 牵着路径线、跟着箭头走(可能走错路或走慢了)。
- 在井里、洞里、纸箱里探出头(够不到边缘)。
- 举着放大镜、望远镜、手电筒看远处。
- 在桌子、纸箱、机器旁搬砖、搭桥、开门、分拣、记录。

## 禁止

- 不要把 LS 画成过度卖萌、夸张表情包、闪亮大眼。
- 不要画成儿童绘本风格的卡通。
- 不要给 LS 复杂服装、配饰、道具堆叠。
- 不要让 LS 只是站在角落里看。
- 不要让 LS 抢走结构表达。
- 不要把 LS 画成写实风格、3D 渲染、复杂光影。
- 不要漂移外形:颜色不能变成棕色或灰色,耳朵不能耷拉,面部白纹不能消失。
- 不要让 LS 笑得太夸张或太多动作戏。

## 判断标准

如果去掉 LS,图的核心隐喻还能完全成立,说明 LS 太装饰了;要重写提示词,让 LS 成为动作主体。

LS 必须正在做"解释这个概念"的那个动作:搬、举、卡、拉、推、扛、守、按、分拣、记录、修补。如果只是微笑站着,就不合格。
